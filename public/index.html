Download a royalty-free mp3 in your browser (e.g. from Pixabay Music), then upload the file here. BPM is measured from the audio itself. This is the reliable path — the link-import below is often blocked by the source site's bot protection.
Even smoother: install the browser extension (see the extension/ folder) to add the track you're viewing on Pixabay with one click — no download or typing.

BPM detection can occasionally read half/double the true tempo — check it sounds right and click the BPM in the library table to fix it if needed.

Add from a Pixabay link (via extension)

Paste a Pixabay music track URL and click Download. The browser extension opens it in a background tab, downloads the mp3 (passing Pixabay's bot check that blocks the server), and adds it here with auto-detected BPM — no manual download. Requires the festival_recap track grabber extension installed and permitted (see the extension/ folder).
